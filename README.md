# SENAI Technical Course Activities and Projects

## 📖 Descrição

Este repositório contém uma coleção completa das atividades, exercícios e projetos desenvolvidos durante o meu curso técnico em Desenvolvimento de Sistemas pelo SENAI. ## 🚀 Projetos Principais

### 1. Sistema de Banco de Imagens
**Localização:** `ATIVIDADES/BANCO-DE-IMAGENS/Banco-de-Imagens/`

Um sistema simples de banco de imagens desenvolvido em Spring Boot com funcionalidades CRUD. Permite armazenar, consultar, atualizar e remover imagens em um banco de dados MySQL.

**Tecnologias:** Java 17, Spring Boot 2.7.5, Spring Data JPA, MySQL, Lombok

### 2. API REST - Cadastro de Pessoas
**Localização:** `ATIVIDADES/ESTUDO_API´S/Aula-API-SpringBoot/`

API RESTful para gerenciamento de cadastro de pessoas, com operações completas de CRUD. Inclui conexão com banco de dados MySQL e estrutura organizada em camadas (Controller, Service, Repository).

**Tecnologias:** Java 17, Spring Boot 3.x, Spring Data JPA, MySQL, Lombok, Maven

### 3. Projeto Web - Autenticação de Usuários
**Localização:** `ATIVIDADES/PROJETOWEB/Projeto-Web/`

Aplicação web para autenticação de usuários com telas de login e cadastro. Inclui funcionalidade de preenchimento automático de endereço utilizando a API ViaCEP.

**Tecnologias:** HTML, CSS, JavaScript, Live Server

### 4. Prova Prática
**Localização:** `ATIVIDADES/PROVA-PRATICA/Prova_pratica/`

Projeto desenvolvido como avaliação prática do curso, demonstrando conhecimentos adquiridos em desenvolvimento de sistemas.

## 💻 Tecnologias Utilizadas

### Linguagens de Programação
- **Java** (versão 17) - Principal linguagem utilizada nos projetos backend
- **JavaScript** - Utilizado no projeto web
- **SQL** - Para consultas e estruturação de bancos de dados

### Frameworks e Bibliotecas
- **Spring Boot** - Framework para desenvolvimento de aplicações Java
- **Spring Data JPA** - Para persistência de dados
- **Lombok** - Para redução de código boilerplate em Java
- **Live Server** - Servidor de desenvolvimento para aplicações web

### Banco de Dados
- **MySQL** - Sistema de gerenciamento de banco de dados relacional

### Ferramentas de Build
- **Maven** - Gerenciamento de dependências e build de projetos Java

## 🏃‍♂️ Como Executar os Projetos

### Pré-requisitos Gerais
- Java 17 instalado
- MySQL Server configurado
- Maven instalado
- Node.js (para projetos web)

### Sistema de Banco de Imagens
```bash
cd ATIVIDADES/BANCO-DE-IMAGENS/Banco-de-Imagens/
mvn clean install
mvn spring-boot:run
```

### API REST - Cadastro de Pessoas
```bash
cd ATIVIDADES/ESTUDO_API´S/Aula-API-SpringBoot/
mvn clean install
mvn spring-boot:run
```

### Projeto Web
```bash
cd ATIVIDADES/PROJETOWEB/Projeto-Web/
npm install
npm start
```
